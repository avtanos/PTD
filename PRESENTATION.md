# OpenCM — Система управления ПТО
## Презентация продукта

---

## Слайд 1: О продукте

**OpenCM** — веб-система для управления документационным сопровождением строительных проектов.

- **Сфера применения**: строительные компании, производственно-технические отделы (ПТО)
- **Стек технологий**: FastAPI, React, TypeScript, SQLite, SQLAlchemy
- **Развёртывание**: Docker, self-hosted или облако

---

## Слайд 2: Основные возможности (1/3)

### Проекты и документация

| Модуль | Описание |
|--------|----------|
| **Проекты** | Управление проектами, заказчик, подрядчик, сроки, конструктивы |
| **Проектная документация** | АР, КР, ВК, НВК, ЭС, ЭМ — версионность и учёт |
| **Исполнительная документация** | Акты, схемы, журналы — учёт и контроль |
| **Исполнительные съемки** | Ведение съёмок геодезистами |
| **Дорожная карта документов** | Планирование и контроль готовности документов по этапам |
| **Изменения проекта** | Согласование изменений и перепланировок с workflow |

## Слайд 3: Основные возможности (2/3)

### Производственная документация

| Модуль | Описание |
|--------|----------|
| **КС-2** | Акт о приёмке выполненных работ с позициями |
| **КС-3** | Справка о стоимости выполненных работ с расчётом НДС |
| **ГПР** | Графики производства работ, задачи и зависимости |
| **ППР** | Проекты производства работ с разделами документации |
| **Учет объёмов работ (ВОР)** | Ведомость объёмов с план/факт контролем |

## Слайд 4: Основные возможности (3/3)

### Закупки, финансы, учёт

| Модуль | Описание |
|--------|----------|
| **Тендеры** | Проведение тендеров, реестр подрядчиков, коммерческие предложения |

| **Дебиторская задолженность** | Учёт, напоминания, взыскание |
| **Отдел продаж** | Коммерческие предложения, согласования с заказчиком |

## Слайд 8: Преимущества системы

- **Единая платформа** — все процессы ПТО в одном интерфейсе  
- **Соответствие нормам** — формы КС-2, КС-3, сметы по принятым стандартам  
- **Интеграция с 1С** — обмен данными с бухгалтерией  
- **Контроль версий** — история изменений документов  
- **Workflow** — согласование заявок и изменений по настраиваемым маршрутам  
- **Адаптивный интерфейс** — светлая/тёмная тема, мобильная версия
